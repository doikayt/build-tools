# Contributing

Thank you for your interest in contributing!

This project's repository contains tooling for multiple platforms. Please read the
relevant guide for the area you are working on.

---

## JavaScript / TypeScript / Node.js Packages

All JavaScript packages live under the [`javascript/`](./javascript/) folder,
managed as a single npm workspace.

## Repo Scripts

Cross-platform utility scripts (not tied to a single `javascript/` package) live under
[`scripts/`](./scripts/README.md), including `lights-out.mjs` for running unattended Claude
Code sessions against a spec file.
